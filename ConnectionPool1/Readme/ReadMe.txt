Weblogic 8.1 / 10.3
---------------------
Put 'weblogic.jar' from %WEBLOGIC_HOME%/server/lib in your classpath and then run this program.


Weblogic 12c
--------------------
You have to run the following command:-
C:\Oracle\Middleware\Oracle_Home\wlserver\server\lib> java -jar wljarbuilder.jar
This will generate a jar file called 'wlfullclient.jar'.
Put this jar file in class path and then run this program.
